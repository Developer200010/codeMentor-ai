# CodeMentor-AI

CodeMentor-AI is a Next.js-based web application designed to help beginner developers identify bugs, flag issues, and receive refactor suggestions to improve their code quality. Built with `create-next-app`, it includes a clean UI, API routes, and a deployment-ready structure.

**Live Demo:** https://code-mentor-ai-one.vercel.app
